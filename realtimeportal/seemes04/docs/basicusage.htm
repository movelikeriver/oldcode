Basic Usage

Introduction

All the work is done from the Admin Control Panel, which can be accessed through the file admin.php in the folder in which you installed Seemes. To get started, go there and log in. (Your password is the one you gave when you installed.)

Overview of the Admin Control Panel

The various tasks you can perform in the Admin Control Panel are listed in the menu on the left-hand side of the page. They shoud be pretty much self-explanatory.

  • Create page: Lets you create a new page.
  • Delete page: Lets you delete an existing page.
  • Edit page: Lets you edit an existing page.
  • Move page: Lets you change the actual filename of an existing page (as opposed to just the title of the page, which can be changed in Edit Page).
  • Edit menu: Lets you edit the menu file, for such things as changing the order of menu items.
  • Image manager: Lets you upload, rename, and delete files in the image directory (default img/).
  • File manager: Lets you upload, rename, and delete files in the general upload directory (default files/).
  • Edit theme: Lets you edit any plain text file in your theme directory.
  • Change settings: Lets you change a few general settings: the website name, copyright line, theme, and password. Note: if you don't enter a new value for a setting (i.e. you leave the field blank), that setting will not be changed.
  • Log out: Lets you log out of your Admin Control Panel. (Highly recommended on public and shared computers.)

General information

The best way to learn to administer Seemes is to play around with it (or wait until better documentation shows up). However, there are a few bits of general information that will help you:

  • Page filename: This is the actual page filename (e.g. mypage). This is what you'll get when you ask for a Page listing. This is what the system uses and should not be confused with...
  • Page name: The "nice" name of the page (e.g. My Page). This is what the user sees.

Installing Themes

This is a pretty simple process. Make sure all the theme files are in their correct place on the website. Now, go into your Admin Control Panel, and the Change settings section. Enter a new theme name (including the extension) and hit Change settings. Now sit back and enjoy your new theme!
