Theming

Introduction

This file contains information regarding making a theme for Seemes. Read this if you want to customize the look of your website.

Designing your theme

Make a space for:

  • the header
  • the menu
  • the main content
  • the footer

The only major limitation is that your main content must come directly after the page title.

Fitting things together

Go through your style sheet (if any) and change the image paths to ../img/yourthemesname/imagename.

Next, rename your HTML file to themename.txt and make the following changes:

  1. The contents of the <title> tag should read:
    <?php echo($websitename); ?> - <?php echo($pagename); ?>
    The bit before the hyphen/dash/negative sign is for the website name, the bit after is for the page name. It's safe to reverse the two or cut one out altogether.
  2. Somewhere in the <head> tag:
    <?php echo($headextra); ?>
  3. And actually inside the brackets of the <body> tag:
    <body<?php echo($bodyextra); ?>>
  4. The header should contain:
    <?php echo($websitename); ?>
    (Unless you have a header image, in which case, that would be the image's alt text.)
  5. The menu*:
    <?php echo($menu); ?>
  6. The content:
    <?php include($page); ?>
  7. And the footer:
    <?php echo($websitecopyright); ?> for the copyright line.
    Also, while not necessary, we would appreciate it very much if you'd put this somewhere in your footer:
    Powered by <a href="http://seemes.org/">Seemes</a> version <?php echo($seemesversion); ?>
  8. If you want to include the "page generated in..." line somewhere in your theme, use the following code:
    <?php echo(timerreturn($starttime)); ?>
    This inserts the time taken (in seconds) from when the page was called to present. Note that it's a good idea to place this at the very end of your theme file so you can measure how long it took to include the page, not just how long it took to generate the menu, etc.

And that's it!

Store all your images in img/yourthemename and your theme file and stylesheet in themes and upload the whole mess to your server.

Now go see Basic Usage for how to install themes.

* - By default, the getmenu function formats all the menu items as anchors inside of list items. If you want it to output something else, you can edit the variable $websitemenuformatting as found in config.php. (The syntax of $websitemenuformatting is described in Advanced Usage.)
