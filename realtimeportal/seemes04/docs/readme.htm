Introduction

Thanks for downloading Seemes CMS (see-mes see-emm-ess), a content management system designed to make the life of a developer easy.

If you're new to Seemes, we recommend reading the installation guide. If you're trying to upgrade your existing Seemes installation, then you'll want to read the upgrading guide.

Please note: A fundimental knowledge of HTML is required to operate a website powered by Seemes, and a basic grasp of CSS would be helpful. Keep in mind that the more you know of each, the better your website can be, no matter what (if any) content management system you use.
