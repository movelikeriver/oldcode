Installation

Installation should be relatively quick and simple. To complete it with minimal difficulty, please follow the steps below:

  1. Upload all the files in the archive to your webserver.
  2. CHMOD the following:
    • The folder that Seemes is in: 0777
    • The folder data/: 0777
    • The folder files/: 0777
    • The folder img/: 0777
    • The folder themes/: 0777
  3. Open (the uploaded copy of) install/install1.php in your browser. Optionally, browsing to the folder you've uploaded Seemes to will give you a link to the installer.
  4. Follow the steps.
  5. Now browse to the root of the folder you've installed to. Your website should be up and running.
  6. Note: It is highly recommended that you delete the folder install/ after the installation has completed. As far as we know, the scripts in it are secure, however, they have not been tested extensively, and should a vulnerability be found in them, an attacker could possibly reset your entire site configuration, including password!
